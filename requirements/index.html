--- layout: default title: "Pathology ScopeCam: System Requirements" --- To view the live video stream, you need:
  • Cable speed or better download bandwidth (to view high quality live image). Any internal University connection should handle this, as well as most connections provide by high speed internet providers and university campuses.
  • A modern web browser with javascript enabled. Internet Explorer 6, Mozilla Firefox, Opera, and Safari should handle this.
  • (Optional for telepathology) Speaker/phone - to communicate with the pathologist.
